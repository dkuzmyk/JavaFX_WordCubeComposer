# JavaFX_WordCubeComposer
Given a word, given up to 6 cubes with 6 faces where each face is a given letter, the program will find all the possible combinations of cube faces to re-create the word.
eg: 
word: vocal
c1: v,c,a,v,o,l 
c2: a,b,b,b,e,z 
c3: h,e,e,a,v,b
c4: l,w,a,a,a,a
c5: a,a,a,a,a,l
c6: a,a,a,a,o,a
